\chapter{はじめに}
\label{chap:intro}

\section{研究背景}
\label{sec:intro-background}

企業の財務構造は，成長・成熟・衰退というライフサイクルに沿って変化する。
\tcite{Dickinson2011} は，営業・投資・財務キャッシュフローのパターンを
企業ライフサイクルの代理変数として用いる手法を提案し，
以降，この枠組みは会計・財務研究の基盤として広く参照されている
\citep{Habib2019survey}。

しかし，既存のライフサイクル研究の多くは，ある時点における企業の分類に焦点を当てており，
企業が年度をまたいでどのように財務構造を変化させるか――すなわち，
クラスター間の遷移――については十分に解明されていない。
\tcite{GonzalezRuiz2021} は K-Means クラスタリングとマルコフ連鎖を用いた遷移分析を
試みたが，対象は 35 社に限定されており，大規模な企業群を対象とした
遷移要因の体系的な分析は行われていない。

日本の上場企業は，有価証券報告書を通じて財務データが体系的に開示されており，
企業の財務構造とその時系列的変化を分析するうえで良質なデータ環境を提供する。
日本企業約 \num{3558} 社・15 年間のパネルデータを活用することで，
米国や欧州を対象とした既存研究を補完する実証分析が可能となる。

近年，機械学習手法の財務分析への応用が急速に進展している。
特に，XGBoost は倒産予測においてAUC 約 94\% を達成し
\citep{Nguyen2023}，SHAP（Shapley Additive Explanations）と組み合わせることで
予測精度と解釈可能性を両立するフレームワークが確立されつつある
\citep{BenJabeur2023, Zhang2023}。
これらの手法を財務クラスター遷移の分析に応用することで，
どの財務変数が遷移を引き起こし，遷移先を決定するかを定量的に解明できる可能性がある。

\section{研究課題}
\label{sec:intro-questions}

本研究では，以下の3つの研究課題（RQ）を設定する。

\begin{enumerate}
  \item[\textbf{RQ1}] 日本上場企業の財務構造は，年度をまたいでどのようなクラスターを形成し，
    各クラスターはどの程度安定的に維持されるか。

  \item[\textbf{RQ2}] クラスター間の遷移（財務構造の変化）は，
    どの財務変数によって引き起こされるか。

  \item[\textbf{RQ3}] 遷移が生じた場合，遷移先クラスターはどの財務変数によって決定されるか。
    また，遷移先によって決定要因は異なるか。
\end{enumerate}

RQ1 は企業の財務構造の安定性とクラスター分類の記述的分析に関する課題であり，
PCA + K-Means クラスタリングとハンガリアン法によるラベル整合によって検討する。
RQ2 および RQ3 は遷移の要因分析に関する課題であり，
XGBoost による予測モデルと SHAP による変数重要度分析によって検討する。

\section{本研究の貢献}
\label{sec:intro-contribution}

本研究は，以下の点で既存研究に対して貢献する。

\begin{enumerate}
  \item \textbf{方法論的貢献}：
    PCA と K-Means クラスタリングにハンガリアン法を組み合わせることで，
    年度間でラベルが整合した財務クラスターを定義する手法を提案する。
    この手法により，5つの財務クラスター
    （C0:中小規模標準型，C1:小規模財務脆弱型，C2:中規模財務優良型，
    C3:大企業型，C4:小規模財務健全型）を識別した。

  \item \textbf{実証的貢献}：
    日本上場企業 \num{3558} 社・15 年間（2010〜2024 年度，\num{49643} 観測）の
    パネルデータを用いて，財務クラスター遷移の実態を記述した。
    年度間の自己遷移率は平均 80.1\% であり，
    企業の財務構造が高い安定性を持つことを明らかにした。
    二値プロビット分析（Pseudo $R^2$ = 0.043）により，
    遷移の有無が財務変数によって説明されることを確認した。

  \item \textbf{予測的貢献}：
    XGBoost と SHAP を用いた2段階予測モデルを構築した。
    遷移の有無を予測する二値分類では AUC = 0.757 を達成し，
    ロジスティック回帰（AUC = 0.630）を大幅に上回った。
    遷移先クラスターの予測ではマクロ AUC = 0.947 を達成した。
    SHAP 分析により，営業利益（Mean $|\mathrm{SHAP}|$ = 0.311）が
    遷移の最重要要因であることを定量的に明らかにした。
\end{enumerate}

\section{論文の構成}
\label{sec:intro-structure}

本論文の構成は以下のとおりである。
第\ref{chap:literature}章では先行研究を概観し，本研究の位置づけを明らかにする。
第\ref{chap:data-method}章ではデータと分析手法を詳述する。
第\ref{chap:results}章では分析結果を報告する。
第\ref{chap:discussion}章では結果の経済的解釈と含意を論じる。
第\ref{chap:conclusion}章では結論と今後の課題を述べる。
