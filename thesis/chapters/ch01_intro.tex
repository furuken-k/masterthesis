\section{研究課題}
\label{sec:intro-questions}

本研究では，以下の3つの研究課題を設定する。

\begin{enumerate}
  \item 日本上場企業の財務構造は，年度をまたいでどのようなクラスターを形成するか。
  \item クラスター間の遷移はどの財務的要因によって引き起こされるか。
  \item 遷移先クラスターはどの財務的要因によって決定されるか。
\end{enumerate}

\section{本研究の貢献}
\label{sec:intro-contribution}

本研究は，以下の点で既存研究に対して貢献する。

\begin{enumerate}
  \item \textbf{方法論的貢献}：
    PCA と K-Means クラスタリングにハンガリアン法を組み合わせることで，
    年度間でラベルが整合した財務クラスターを定義する手法を提案する。

  \item \textbf{実証的貢献}：
    日本上場企業 \num{3558} 社・15 年間（2010〜2024 年）のパネルデータを用いて，
    財務クラスター遷移の実態を記述し，遷移要因を統計的・機械学習的手法で解明する。

  \item \textbf{予測的貢献}：
    XGBoost と SHAP を用いて，どの財務変数が遷移を引き起こし，
    どの変数が遷移先を決定するかを定量的に明らかにする。
\end{enumerate}
